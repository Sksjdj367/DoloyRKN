## DoloyRKN - experimental tool for analyzing and experimenting with Deep Packet Inspection (DPI) mechanisms.

The author does not encourage or condone violation of local laws, network policies, or terms of service.
Users are responsible for ensuring compliance with appliccable regulations.

### **Installation**:
Install latest version from [Releases](https://github.com/Sksjdj367/DoloyRKN/releases) page.

### **Supported systems**:
* Windows 7 and above (64-bit)
* Linux x86_64 (64-bit)

### **Documentation**:
Detailed documentation available in [Docs](https://github.com/Sksjdj367/DoloyRKN/tree/main/docs) folder.
