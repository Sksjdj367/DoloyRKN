## DoloyRKN - утилита для обхода блокировок в России.

## Использование
Скачайте последний релиз со страницы [релизов](https://github.com/Sksjdj367/DoloyRKN/releases).
Запустите скрипт s_run для запуска обхода, готово.

## Способности обхода
Программа обходит блокировку всех сервисов, заблокированных РКН, включая:
 - Youtube
 - Discord
 - Instagram
 - Whatsapp
 - Viber
 - X (Twitter)
 - Facebook
 - LinkedIn

## Назначение скриптов
Скрипт s_run содержит необходимые настройки для программы, позволяющие обходить блокировки.

Для Windows доступны скрипты:
s_enable_autoload - добавляет программу в автозагрузку.
s_disable_autoload - убирает программу из автозагрузки.

### Поддерживаемые Системы:
* Windows 7 и выше (64-бита)
* Linux 3.6 и выше (64-бита)
